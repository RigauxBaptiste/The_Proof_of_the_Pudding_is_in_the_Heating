ReadMe file for the DAM_price.csv datafile.

### Description

These datasets collect the 60-min day-ahead market prices on the spot market for electricity in Belgium for the years 
- 2022
- 2023 
- and 2024 (until July 2024)

### Source of the data

The data comes from the ENTSO-E transparency platform: https://transparency.entsoe.eu/transmission-domain/r2/dayAheadPrices/show?name=&defaultValue=false&viewType=TABLE&areaType=BZN&atch=false&dateTime.dateTime=01.03.2022+00:00|CET|DAY&biddingZone.values=CTY|10YBE----------2!BZN|10YBE----------2&resolution.values=PT60M&dateTime.timezone=CET_CEST&dateTime.timezone_input=CET+(UTC+1)+/+CEST+(UTC+2).

### Notes

The data have been put into one single output file ready to use: DAM_prices.csv.
Unit after the cleaning: €/kWh (€/MWh in the original files from ENTSO-E).
